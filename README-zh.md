# 当贝遥控器

[English Version](README.md) | 中文版本

一个局域网专属的当贝设备遥控器，手机通过浏览器访问即可控制。

**当前版本：v1.0.0**

## 功能特点

- 移动端优化UI，圆形方向键设计
- 启动时自动尝试连接已知设备
- 扫描优先尝试已知设备，失败后再全网扫描
- 手机浏览器访问，无需安装APP
- 支持所有常用遥控器按键（含快速关机、重启）
- 支持长按重复发送（方向键、音量键）
- 防误触设计（滑动出按钮区域取消）
- 配置自动保存，重启后保留
- 音量0-15格调节
- WebSocket持久连接，减少握手开销
- 状态缓存，快速响应状态查询
- 音量滑块拖拽状态控制，防止操作冲突
- 网络扫描使用ipaddress模块，支持自定义网段
- Semaphore并发控制
- 设备状态监控：在线时3秒检查一次，离线时10秒检查一次
- 每日首次设备上线后自动重启（延迟5秒确保初始化完成）

## 快速开始

### 方式一：使用uv（推荐）

1. 安装uv（如果未安装）：

**Windows:**
```powershell
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

**Linux/Mac:**
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

2. 运行项目：
```bash
uv run python server.py
```

### 方式二：使用启动脚本

**Windows:** 双击 `start.bat`

**Linux/Mac:**
```bash
chmod +x start.sh
./start.sh
```

### 方式三：手动启动

1. 安装依赖：
```bash
pip install aiohttp websockets
```

2. 运行服务：
```bash
python server.py
```

## 使用说明

1. 确保运行本程序的电脑和当贝投影设备在同一局域网
2. 启动服务后，在手机浏览器访问显示的地址（如 `http://192.168.1.100:8080`）
3. 未连接时点击顶部状态栏开始扫描
4. 已连接时点击顶部状态栏查看设备信息
5. 开始使用遥控器

## 配置

- 默认端口：8080
- 配置文件：`.data/config.json`（保存已知设备、音量、最后重启日期）
- 可通过 `--port` 参数修改端口：`uv run python server.py --port 9090`
- 自定义扫描网段：在配置文件中添加 `"scan_network": "192.168.1.0/24"` 字段（可选）

## 执行流程

1. **启动时**：尝试连接配置文件里的已知设备
2. **连不上**：状态设为offline，等待用户触发扫描
3. **用户点击顶部栏（未连接）**：先尝试已知设备，失败后全网扫描
4. **用户点击顶部栏（已连接）**：打开底部设备列表
5. **收到指令**：直接发送给设备
6. **发送失败**：标记设备离线，弹出离线确认框
7. **每日首次上线**：检测到设备从离线变在线后，等待5秒自动重启一次

## 按键说明

| 按键 | 功能 |
|------|------|
| 电源 | 开关电源 |
| 快速关机 | 发送关机序列 |
| 重启 | 发送重启序列 |
| 侧边键 | 侧边菜单 |
| 音量+/- | 音量增减（支持长按） |
| 方向键 | 上下左右（支持长按） |
| OK | 确认 |
| 主页 | 返回主页 |
| 菜单 | 打开菜单 |
| 返回 | 返回上一级 |
| 找遥控器 | 让设备发出提示音 |

## 已知限制

由于当贝设备协议限制，无法直接读取设备当前的绝对物理音量，只能发送相对增减指令。如果使用实体遥控器调整音量，网页端的音量显示会与设备物理音量不同步，建议尽量统一使用网页端控制音量。
